Introduction to CSS (Cascading Style Sheets)

CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe how HTML elements should appear on a webpage. HTML only defines the structure of the content, while CSS controls how that content looks. Using CSS, you can change colors, layouts, fonts, spacing, backgrounds, borders, animations, and much more.

Why Do We Use CSS?

  • Improves appearance: Makes webpages look professional, clean, and attractive.
  • Faster development: A single CSS file can style multiple pages.
  • Separation of structure and design: HTML handles structure, CSS handles style.
  • Responsive design: Helps create mobile, tablet, and desktop layouts.
  • Reusability: CSS rules can be reused across multiple elements or pages.

How CSS Works (The “Cascading” Concept)

The term cascading means that multiple CSS rules can apply to the same element. The browser decides which rule takes effect based on three factors:

  • Specificity: Inline > ID > Class > Element
  • Order: Later rules override earlier ones
  • Importance: !important overrides everything
Example: If both p { color: blue; } and #para { color: red; } target the same paragraph, the ID selector wins, so the text becomes red.

Ways to Add CSS in HTML

1. Inline CSS

Used inside the HTML tag using the style attribute. Best for quick testing, not recommended for final designs.

<p style="color: blue; font-size: 20px;">This is inline CSS</p>

2. Internal CSS

Placed inside the <style> tag in the HTML <head> section. Good for single-page styling.

<style>
    h2 {
        color: green;
        text-align: center;
    }
</style>

3. External CSS

Stored in a separate file (e.g., style.css) and linked using <link>. Best for multi-page websites.

<link rel="stylesheet" href="styles.css">

Basic CSS Selectors

1. Element Selector

Targets all elements of a specific type.

p { color: red; }

2. Class Selector

Targets elements with a specific class. Starts with a dot .

.box { background: yellow; padding: 10px; }

3. ID Selector

Targets the element with a unique ID. Starts with #.

#header { font-size: 30px; }

4. Universal Selector

Applies to all elements.

* { margin: 0; padding: 0; }

5. Group Selector

Targets multiple elements at once.

h1, h2, p { font-family: Arial; }

CSS Properties Examples

Text Properties

  • color: Sets text color
  • font-size: Changes size of text
  • text-align: Aligns text (left, center, right)
  • font-weight: Makes text bold

Box Model Properties

  • margin: Space outside the element
  • padding: Space inside the element
  • border: Border around an element
  • width / height: Size of the element

Example CSS Code

h1 {
    color: #3498db;
    text-align: center;
    padding: 10px;
}

p {
    font-size: 18px;
    color: #555;
}

Conclusion

CSS is essential for modern web development. It separates design from content, making websites faster, easier to maintain, and visually better. Once you understand CSS selectors, properties, and the box model, you can create beautiful and responsive webpages effortlessly.
